Personal Finance Tracker - README
Overview
A comprehensive web application for tracking personal finances, enabling users to monitor and analyze their income and expenses with advanced categorization and filtering options.

Features
Core Features
Transaction Management: Add, view, and delete income/expense transactions

Detailed Categorization:

Pre-defined categories for income and expenses

Custom category creation with color coding

Category management interface

Advanced Filtering & Search:

Filter by transaction type (Income/Expense)

Filter by category

Date range filtering (Today, Week, Month, Year)

Text search in descriptions and notes

Data Visualization:

Income vs Expenses bar chart (monthly breakdown)

Expense breakdown pie chart by category

Data Export: Export all transactions to CSV format

Data Persistence: All data stored locally in browser storage

Technical Features
Responsive Design: Optimized for desktop, tablet, and mobile devices

No Dependencies: Pure JavaScript implementation

Offline Capable: Works completely offline

Setup Instructions
Prerequisites
Modern web browser (Chrome, Firefox, Safari, Edge)

No additional software or dependencies required

Installation
Download the index.html file

Open the file in your web browser:

Double-click the file, or

Right-click → Open With → Your preferred browser
iew Dashboard: The main screen shows your current balance, total income, and total expenses

Navigate Tabs: Use the tabs to switch between different views:

Transactions: View and manage your transaction history

Charts: Visualize your financial data

Add Transaction: Create new income/expense entries
Exporting Data
Click "Export CSV" in the header

A CSV file will be downloaded with all your transactions

File name format: finance-tracker-YYYY-MM-DD.csv

Viewing Charts
Click the "Charts" tab to see visualizations:

Income vs Expenses: Monthly comparison bar chart

Expense Breakdown: Pie chart showing expense distribution by category
Data Storage
Local Storage
All data is stored in your browser's local storage

Data persists between browser sessions

No data is sent to external servers

Backup Recommendations
Regularly export your data using the CSV export feature

Save exported CSV files to cloud storage or external drives

Browser Compatibility
Chrome 50+

Firefox 45+

Safari 10+

Edge 12+
All devices (Responsive)
Common Issues
Data Not Saving

Ensure cookies and local storage are enabled in your browser

Check if you're in private/incognito mode (some browsers restrict storage)

Charts Not Displaying

Ensure JavaScript is enabled

Check browser console for any error messages

Filters Not Working

Clear browser cache and reload the application

Try using a different browser
Privacy & Security
All data remains on your local device

No data is transmitted to external servers

No user tracking or analytics

Export feature allows complete data ownership
